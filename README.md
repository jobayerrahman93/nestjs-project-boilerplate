
## Description

## This project has been done, so that any project can start quickly with user authentication




### This project has many features below like:

- Routing Module
- Guard
- Custom Global Error Handler
- Prisma
- Mysql
- JWT
- env config service
- Global prisma service


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
